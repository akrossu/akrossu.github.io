Medieval Modpack installation steps:

IMPORTANT: If at any point, your modpack has failed to install, completely delete and reinstall the modpack and start from the beginning.

  1. - Install the CurseForge App and the Medieval Modpack.
  2. - Make sure to allocate 8 - 10 gigabytes of RAM.
  3. - Delete and replace your entire mods folder with the updated mod list.
  4. - Start your game before replacing any other files.
      5. - Please relace your existing files with these.
